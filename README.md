# 🏥 Hospital Emergency Room Dashboard

An interactive multi-page Power BI dashboard developed to analyze emergency room operations, patient admissions, waiting times, patient satisfaction, department referrals, and demographic trends.

The dashboard transforms hospital data into actionable insights using **Power Query, Data Modeling, DAX, time intelligence, and interactive visualizations** to help healthcare professionals monitor performance and improve patient care.

# 📌 Project Overview

This project provides a centralized analytical solution for monitoring hospital emergency room performance and understanding patient behavior.

The dashboard enables users to:

* Monitor key healthcare KPIs.
* Analyze patient trends on monthly and yearly levels.
* Track average waiting times and patient satisfaction.
* Evaluate admission and referral patterns.
* Analyze patient demographics.
* Identify peak hospital hours and busiest days.
* Explore detailed patient records.

---

# 🎯 Business Objectives

The dashboard was developed to analyze:

* Overall emergency room performance.
* Monthly patient admission trends.
* Average patient waiting time.
* Patient satisfaction scores.
* Admission and non-admission rates.
* Department referral patterns.
* Age-group, gender, and race distribution.
* Day-wise and hour-wise patient volume.
* Detailed patient records.

# 🔍 Sample Insights

The following observations are based on the dashboard preview:

* More than **9,200 patients** visited the emergency room during the selected period.
* The average waiting time was approximately **35 minutes**.
* Patient satisfaction remained stable with a score of **5**.
* Around **3,800 patients** were referred to different departments.
* Most patients belonged to the **20–39 age group**.
* The highest number of referrals came from **General Practice and Orthopedics**.
* Patient traffic varied significantly by day and hour.

Dashboard values update dynamically based on the selected month and date range.
